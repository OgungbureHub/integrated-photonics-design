# Integrated Photonics Design Portfolio

This repository serves as my **ongoing portfolio** of projects, simulations, layouts, and design work from the **Advanced Master of Science in Silicon Photonics** at Ghent University (2025–2026).

It showcases my practical experience in Photonic Integrated Circuit (PIC) design, simulation, layout, fabrication support, and device characterization. The repository will be **regularly updated** as I complete new projects, add measurement results, and refine existing work.

## Current Projects

### Ring Modulators
- Design and simulation of ring-based electro-optic modulators
- Integration of PZT/PLZT thin films for high-speed modulation
- Optical and electro-optic performance analysis (simulation + ongoing experimental characterization)

### Serially Coupled Vernier Ring Resonator Filter
- Parameterized design flow for serially coupled Vernier ring resonators
- Full Python + KLayout + PDK workflow from schematic to DRC-compliant GDSII layout
- Circuit-level simulation and laboratory optical/electrical validation

### Thermo-Optic Phase Tuner (SiN Platform)
- Design and layout of thermo-optic phase shifters on silicon nitride platform
- Thermal and optical simulation using COMSOL and Lumerical
- KLayout mask layout implementation

### Additional PIC Design Work
- Mask layouts and tape-out preparation
- IPKISS-based photonic circuit design
- Optical simulations (FDTD, MODE, circuit-level)

## Tools & Technologies

- **Simulation:** Lumerical (FDTD/MODE), COMSOL Multiphysics, Keysight ADS
- **Design & Layout:** KLayout, IPKISS PDK, Python scripting for parameterized designs
- **Characterization:** Optical/electrical measurements, VNA calibration (open/short/load/through), PI curves, small-signal modulation, RIN analysis
- **Programming:** Python (data processing, automation, ML pipelines), MATLAB/Simulink

## Key Skills Demonstrated

- End-to-end PIC design flow: concept → simulation → layout → validation
- Parameterized design and automation using Python and PDKs
- Device fabrication awareness and hands-on characterization in cleanroom environment (INTEC)
- Comparison of simulated vs. measured performance of photonic components
- Thin-film integration (PZT/PLZT) for electro-optic applications
